# 防抖(Debounce)

当持续触发事件时,一定时间段内没有再触发事件,事件处理函数才会执行一次,如果设定的时间到来之前,又一次触发事件,就重新开始延时。

  1. search 搜索联想
  2. window 触发 resize
  3. 防止重复提交

# 节流(Throttle)

当持续触发事件时,保证一定时间段内只调用一次事件处理函数。

  1. 鼠标不断点击触发
  2. 监听滚动事件
